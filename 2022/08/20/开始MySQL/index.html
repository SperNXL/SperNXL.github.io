开始MySQL

Author: Arno Dorian Date: 八月 20, 2022  0:18:57 Category: MySQL
安装MySQL

略。。。

启动MySQL

CMD以管理员身份输入

net stop mysql 停止

net start mysql 启动

客户端连接

方式一：MySQL提供的命令行工具

输入密码即可启动

方式二：系统自带的命令行工具执行指令

mysql [-h 127.0.0.1] [-P3306] -u root -p

其中：-h 是指连接的那个ip ， -P后是指连接的哪个端口 ，这两个可以省略

​ -u root: 以root用户进行连接， -p：密码

注：使用这种方式需要配置PATH环境变量

配置PATH环境变量

我的电脑->属性->高级->环境变量->系统变量->path

新建，将mysql路径粘贴进去 默认为C:\Program Files\MySQL\MySQL Server 8.0\bin\ 然后确定
